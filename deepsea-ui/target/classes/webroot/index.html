    Welcome to Deepsea Insurance

    This is a simple (at the moment) project to recreate an Open Source Insurance application based on Vertx and Openshift. A long way to go yet :)

    And special thanks to Clement Escoffier and Eric Zhao for the initial tutorials that pointed me this way.

    Thanks, Andy

    About Deep Sea

    This project was created as an openshift learning project using kubernetes, java and vertx. This is currently an experimental project so many things do not work.

    Currently it is using Redis for Pub/sub, mongo for document writes, mysql for reviewable data (with redis cache now implementing), infinispan for clustering. Also need to write tests for everything.
